# Changelog

Усі зміни проєкту документуються у цьому файлі.

## [0.1.0] - 2026-02-22

### Додано

- Каталог товарів з повнотекстовим пошуком та фільтрацією
- Оптово-роздрібна система ціноутворення з персональними цінами
- Кошик з об'єднанням після авторизації
- Система замовлень зі статусною моделлю (new_order -> processing -> confirmed -> paid -> shipped -> completed)
- Адмін-панель (18 сторінок): замовлення, товари, категорії, користувачі, імпорт, аналітика, налаштування
- Кабінет клієнта з різною навігацією для роздрібних та оптових покупців
- Імпорт товарів з Excel з автоматичною валідацією та створенням категорій
- Telegram-бот з каталогом, пошуком, акціями та нотифікаціями
- 3 теми оформлення згідно ТЗ: «Свіжість та Органіка», «Кристальна чистота», «Домашній затишок»
- Динамічний контент на головній: банери з CMS, USP-блок, логотипи брендів
- Плаваюча кнопка "Замовити дзвінок" з формою зворотного зв'язку
- Покращена 404 сторінка з пошуком та популярними категоріями
- Кнопка шерингу в Instagram Stories
- Стилі для друку замовлень та рахунків
- Генерація PDF: рахунки-фактури, видаткові накладні, комерційні пропозиції
- Push-сповіщення через Service Worker
- Реферальна та бонусна програми
- Інтеграція з Новою Поштою та Укрпоштою
- Оплата через LiqPay та Monobank
- Unit-тести (467 тестів) з покриттям сервісів, валідаторів, API-роутів та утиліт
- E2E тести на Playwright
